Help

  • A new element creation - the key TAB
  • Element switching - arrow keys
  • Element deletion - the key DEL
  • Element selection occurs by hovering mouse cursor over the element and left click
  • Text editing - the key F2 or double-click on the element
  • Drag and drop of an element- press the left mouse button on the element and drag it to a new position.
